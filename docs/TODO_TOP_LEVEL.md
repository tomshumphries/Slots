# Top-Level Goals

List of the next 5 overall goals to get done. Rough guidance and direction for the tool.

## Current Status: MVP Complete
Basic slot machine is playable with menu, balance, and win detection.

---

1. **Enhance Cascade/Match Mechanics**
   - Implement Bejeweled-style cascading where matched symbols disappear and new ones fall in
   - Add chain reaction bonuses for consecutive matches
   - Visual feedback for cascades (symbols falling, particles)

2. **Implement Proper Bonus Feature**
   - Design bonus round triggered by 3x special symbols (currently just gives £20)
   - Create bonus mini-game (free spins, pick-a-prize, wheel spin)
   - Add bonus win animations and sounds

3. **Balance & Tuning**
   - Review symbol distribution and payout rates
   - Implement RTP (Return to Player) calculation
   - Add variable bet amounts (£0.50, £1, £2, £5)
   - Track win/loss statistics

4. **Polish & Visual Improvements**
   - Add sound effects (spin, win, bonus)
   - Improve animations (smoother spins, better win celebrations)
   - Add particle effects for big wins
   - Consider custom graphics instead of emojis

5. **Extended Features**
   - Add persistent storage (save balance between sessions)
   - Win history / statistics page
   - Settings menu (sound on/off, etc.)
   - Achievement system
